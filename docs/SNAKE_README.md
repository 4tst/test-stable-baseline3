## 🎮 如何运行游戏

### 方式一：交互模式
```bash
python snake.py
```
然后根据提示选择模式（1/2/3）

### 方式二：命令行参数
```bash
# 训练AI
python snake.py train

# 测试AI
python snake.py test

# 人类玩家模式
python snake.py play
```

### 方式三：运行测试
```bash
python test_snake.py
```

## 🎯 游戏模式说明

### 1. 训练AI (train)
- 使用 PPO 算法训练 AI
- 训练 100,000 步
- 模型保存为 `ppo_snake_v1.zip`
- 日志保存在 `./logs/` 目录

### 2. AI测试 (test)
- 加载已训练的 AI 模型
- 图形界面展示 AI 游戏过程
- 需要先运行训练模式

### 3. 人类玩家模式 (play)
- 使用方向键控制蛇
- ↑ ↓ ← → 控制移动方向
- 吃到红色食物得分
- 撞墙或撞自己游戏结束

## 🔧 技术细节

### 环境配置
- **网格大小**: 20x20
- **动作空间**: 3种动作（直走、左转、右转）
- **观察空间**: 8维向量
  - 蛇头坐标 (X, Y)
  - 食物坐标 (X, Y)
  - 危险检测（前方、后方）

### 奖励机制
- 吃到食物: +10 分
- 死亡: -10 分
- 每步时间惩罚: -0.1 分
- 超时死亡: -5 分

### 依赖库
- gymnasium >= 0.29.0
- pygame >= 2.5.0
- stable-baselines3 >= 2.0.0
- numpy >= 1.24.0
