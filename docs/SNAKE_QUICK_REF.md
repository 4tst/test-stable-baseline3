# 贪吃蛇游戏 - 快速参考

## 🚀 快速启动

```bash
python snake.py play
```

## 🎮 控制方式

| 按键 | 功能 |
|------|------|
| ↑ | 向上移动 |
| ↓ | 向下移动 |
| ← | 向左移动 |
| → | 向右移动 |
| 任意键 | 游戏结束后重新开始 |
| ESC | 退出游戏 |

## 📋 游戏流程

```
1. 启动游戏
   ↓
2. 蛇自动向右移动（初始方向）
   ↓
3. 【立即按方向键】改变方向 ← 关键步骤！
   ↓
4. 控制蛇吃红色食物
   ↓
5. 撞墙或撞自己 → Game Over
   ↓
6. 屏幕显示 "Press Any Key to Restart"
   ↓
7. 按任意键开始新游戏
   ↓
8. 回到步骤 2
```

## ⚠️ 重要提示

### ❗ 游戏开始时
- **蛇的初始方向是向右**
- **你需要立即按方向键**来改变方向
- 如果不按键，蛇会继续向右直到撞墙

### ❗ 游戏结束时
- **不会自动重新开始**
- 屏幕会显示 "Press Any Key to Restart"
- **按任意键后才会开始新游戏**
- 你有充足的时间准备

## 💡 常见问题

### Q: 为什么蛇总是向右走？
A: 这是初始方向。游戏开始后**立即按方向键**即可改变。

### Q: 为什么游戏结束后不自动开始？
A: 为了给你充足的准备时间。按任意键后才会开始新游戏。

### Q: 如何避免重复撞墙？
A: 
1. 游戏开始后立即按方向键
2. 游戏结束后等待你准备好再按键重新开始

### Q: 如何提高得分？
A:
- 提前规划路径
- 尽量沿着墙壁或自身形成螺旋
- 注意食物位置
- 避免把自己围住

## 🎯 游戏目标

- 控制蛇吃到红色食物
- 每吃一个食物，蛇身增长，得分 +1
- 避免撞墙或撞到自己
- 尽可能获得高分！
